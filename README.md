# Order Inventory Management

A Spring Boot application for managing products, orders, and inventory.

## Features

* Product management
* Inventory management
* Order management
* Stock validation
* Exception handling

## Technologies Used

* Java
* Spring Boot
* Spring MVC
* Thymeleaf
* Maven
* MySQL

## Architecture

* Controller Layer
* Service Layer
* Repository Layer
* Entity Layer
* Exception Handling
